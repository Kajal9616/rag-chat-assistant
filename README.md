# rag-chat-assistant
RAG-based Chat Assistant using FastAPI, FAISS, and Streamlit for intelligent question answering from custom dataset.

## 📌 Project Description
This is a Retrieval-Augmented Generation (RAG) based Chat Assistant that uses FastAPI, FAISS, and Streamlit to answer questions from a custom dataset.

---

## ⚙️ Tech Stack
- FastAPI (Backend)
- FAISS (Vector Search)
- LangChain
- Streamlit (Frontend)
- Google Gemini / LLM

---

## 🚀 Features
- Ask questions in natural language
- Semantic search using FAISS
- LLM-based intelligent answers
- Simple UI using Streamlit
